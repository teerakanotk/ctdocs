# ขั้นตอนการดาวน์โหลด CT Templates

1. เปิดเว็บเบราว์เซอร์ แล้วเข้าไปที่ `https://<IP_ADDRESS>:8006`

2. ไปที่เมนู Datacenter > Node > local > CT Templates

   ![lxc-template-1](/images/proxmox-ve/lxc-template-1.png)

3. คลิกปุ่ม Templates ที่ด้านบนของหน้า

4. เลือกเทมเพลตที่ต้องการใช้งาน แล้วคลิก Download

   ![lxc-template-2](/images/proxmox-ve/lxc-template-2.png)

5. เมื่อดาวน์โหลดเสร็จ เทมเพลตจะปรากฏอยู่ในรายการ

> 💡 หมายเหตุ: หากไม่มีอินเทอร์เน็ต หรือ repository ถูกบล็อก อาจไม่สามารถดาวน์โหลดเทมเพลตได้
