# CTDOCS

เนื้อหาภายในเว็บไซต์ครอบคลุมการติดตั้งและจัดการแอปพลิเคชันต่าง ๆ ที่จำเป็นสำหรับการเริ่มต้นโปรเจค รวมถึงเทคนิคและคำแนะนำสำหรับผู้เรียนหรือผู้พัฒนา

![hero](/public/img/hero.png)

## การใช้งานเบื้องต้น

- เปิดเว็บไซต์ [https://ctdocs.vercel.app/docs](https://ctdocs.vercel.app/docs)
- เลือกหัวข้อหรือโปรแกรมที่สนใจจากเมนู
- อ่านและปฏิบัติตามขั้นตอนในคู่มือได้ทันที

---

เว็บไซต์นี้พัฒนาด้วย [Nextra.js](https://nextra.site/) ซึ่งเป็นเฟรมเวิร์กสำหรับสร้างเว็บไซต์ Documentation ที่ทันสมัยและใช้งานง่าย
