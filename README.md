## เกี่ยวกับ CTDOCS

CTDOCS เป็นเว็บไซต์ Document และสื่อการสอนที่ออกแบบมาเพื่อช่วยให้ผู้เริ่มต้นโปรเจคสามารถเข้าถึงไกด์และคู่มือการใช้งานต่างๆ ได้อย่างง่ายดาย  
เนื้อหาภายในเว็บไซต์ครอบคลุมการติดตั้งและจัดการแอปพลิเคชันต่าง ๆ ที่จำเป็นสำหรับการเริ่มต้นโปรเจค รวมถึงเทคนิคและคำแนะนำสำหรับผู้เรียนหรือผู้พัฒนา

เว็บไซต์นี้พัฒนาด้วย [Nextra.js](https://nextra.site/) ซึ่งเป็นเฟรมเวิร์กสำหรับสร้างเว็บไซต์ Documentation ที่ทันสมัยและใช้งานง่าย

## ฟีเจอร์หลัก

- คู่มือและสื่อการสอนที่อัปเดตอย่างต่อเนื่อง
- ระบบจัดการเนื้อหาง่ายต่อการขยาย เพิ่มเติมหัวข้อและบทเรียนใหม่ๆ
- รองรับการค้นหาข้อมูลภายในเว็บไซต์
- โครงสร้างไฟล์และเนื้อหาที่จัดเรียงอย่างเป็นระบบและอ่านง่าย
- รองรับการทำงานแบบ Static Site Generation เพื่อโหลดเร็ว

## การใช้งานเบื้องต้น

1. เปิดเว็บไซต์ [https://ctdocs.vercel.app/docs](https://ctdocs.vercel.app/docs) เพื่อเรียกดูเอกสารและคู่มือต่าง ๆ
2. เลือกหัวข้อหรือโปรแกรมที่สนใจจากเมนู
3. อ่านและปฏิบัติตามขั้นตอนในคู่มือได้ทันที
