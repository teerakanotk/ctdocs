# Nginx Proxy Manager

เครื่องมือโอเพนซอร์สที่ช่วยให้คุณสามารถจัดการ **Reverse Proxy** ได้ง่ายผ่าน **Web GUI** โดยไม่ต้องเขียนไฟล์ config ของ Nginx ด้วยตนเอง รองรับการตั้งค่า SSL แบบ Let's Encrypt, การจัดการโดเมน, Redirect, และอื่น ๆ

## Reverse Proxy คืออะไร

Reverse Proxy คือเซิร์ฟเวอร์ตัวกลางที่รับคำขอ (Request) จากผู้ใช้งาน แล้วส่งต่อไปยังเซิร์ฟเวอร์ปลายทาง (Backend) จากนั้นจะรับผลลัพธ์กลับมาและส่งให้ผู้ใช้งานอีกที โดยที่ผู้ใช้งานไม่ทราบว่าเบื้องหลังมีเซิร์ฟเวอร์หลายตัวอยู่

ตัวอย่างที่พบบ่อย เช่น:
- `example.com` → ส่งไปยัง web server
- `api.example.com` → ส่งไปยัง API server
- `admin.example.com` → ส่งไปยังหน้า dashboard

## ความต้องการเบื้องต้น

- ติดตั้ง Docker
- ติดตั้ง Docker Compose

> หากยังไม่ได้ติดตั้ง Docker สามารถดูวิธีติดตั้งได้ [ที่นี่](/docs/docker)

## ขั้นตอนการติดตั้ง

<Steps>

### สร้างโฟลเดอร์โปรเจกต์

```bash
mkdir nginx-proxy-manager
cd nginx-proxy-manager
```

### สร้างไฟล์ docker-compose.yml

```bash
sudo nano docker-compose.yml
```

จากนั้นวางเนื้อหาด้านล่าง:

```bash filename="~/nginx-proxy-manager/docker-compose.yml"
services:
  app:
    container_name: nginx-proxy-manager
    image: jc21/nginx-proxy-manager:latest
    restart: unless-stopped
    ports:
      - 80:80
      - 81:81
      - 443:443
    volumes:
      - ./data:/data
      - ./letsencrypt:/etc/letsencrypt
```

### รัน Nginx Proxy Manager

```bash
sudo docker compose up -d
```

Docker จะดาวน์โหลด image และเริ่มต้น container อัตโนมัติ

```bash
root@npm:~/nginx-proxy-manager# sudo docker compose up -d

[+] Running 2/2
 ✔ Network nginx-proxy-manager_default  Created             0.1s 
 ✔ Container nginx-proxy-manager        Started             0.4s 
```

## เริ่มต้นใช้งาน

1. เปิดเบราว์เซอร์ และเข้าไปที่ `http://<IP_ADDRESS>:81`
    > ตัวอย่างเช่น `http://192.168.1.100:81`

    ![install-1](/img/nginx-proxy-manager/install-1.png)

2. ล็อกอินด้วยข้อมูลเริ่มต้น

    - Email: `admin@example.com`
    - Password: `changeme`

3. ระบบจะให้คุณเปลี่ยนข้อมูลบัญชี

    - เปลี่ยนอีเมล

        ![install-2](/img/nginx-proxy-manager/install-2.png)

    - เปลี่ยนรหัสผ่าน

        ![install-3](/img/nginx-proxy-manager/install-3.png)

4. เข้าสู่หน้าหลักของ Dashboard
