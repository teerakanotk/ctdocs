# Proxmox

แพลตฟอร์ม virtualization ที่รวมเทคโนโลยี KVM (Kernel-based Virtual Machine) และ LXC (Linux Containers) เข้าด้วยกัน ทำให้สามารถรันทั้งเครื่องเสมือน (Virtual Machines - VMs) และคอนเทนเนอร์ (Containers) บนเซิร์ฟเวอร์เดียวกันได้อย่างมีประสิทธิภาพ โดยมีจุดเด่นคือการจัดการผ่านเว็บอินเทอร์เฟซที่ใช้งานง่าย และการสนับสนุนการทำงานแบบคลัสเตอร์เพื่อความพร้อมใช้งานสูง (High Availability)

Proxmox VE ถูกพัฒนาขึ้นบนฐานของ Debian Linux ซึ่งทำให้มีความเสถียรและสามารถปรับแต่งได้ตามความต้องการ อีกทั้งยังเป็นซอฟต์แวร์ฟรีที่มีชุมชนผู้ใช้งานขนาดใหญ่และมีตัวเลือกการสมัครสมาชิกเพื่อรับการสนับสนุนระดับองค์กร

## คุณสมบัติหลัก

1. Virtualization และ Containerization
    - รองรับ KVM สำหรับการรัน VMs ที่สามารถติดตั้งระบบปฏิบัติการได้หลากหลาย เช่น Windows, Linux, หรือ BSD
    - รองรับ LXC สำหรับคอนเทนเนอร์ที่มีน้ำหนักเบา เหมาะสำหรับการรันแอปพลิเคชันที่ต้องการทรัพยากรน้อย

2. Web-based Management Interface
    - Proxmox VE มีหน้าเว็บที่ใช้งานง่าย ช่วยให้ผู้ดูแลระบบสามารถจัดการ VMs, คอนเทนเนอร์, สตอเรจ, และเครือข่ายได้จากที่เดียว
    - รองรับการเข้าถึงจากระยะไกลและการจัดการแบบหลายผู้ใช้ (Multi-user)

3. Clustering และ High Availability
    - รองรับการสร้างคลัสเตอร์ของเซิร์ฟเวอร์หลายเครื่อง เพื่อให้สามารถโยกย้าย VMs หรือคอนเทนเนอร์ระหว่างโหนดได้โดยไม่หยุดชะงัก
    - รองรับการทำงานแบบ High Availability เพื่อให้บริการไม่หยุดชะงักในกรณีที่โหนดใดโหนดหนึ่งล้มเหลว

4. Storage Management
    - รองรับระบบสตอเรจหลากหลาย เช่น ZFS, LVM, NFS, Ceph, และ GlusterFS
    - มีความสามารถในการทำ snapshot, backup, และ replication เพื่อความปลอดภัยของข้อมูล

5. Backup Tools

   - มีระบบสำรองข้อมูลในตัวที่สามารถกำหนดตารางเวลาและกู้คืนข้อมูลได้อย่างง่าย
   - รองรับการสำรองข้อมูลไปยังสตอเรจภายนอกหรือเซิร์ฟเวอร์ระยะไกล

## ดาวน์โหลด ISO

1. เปิดเว็บเบราว์เซอร์และไปที่

   ```
   https://www.proxmox.com/en/downloads
   ```

2. เลือกเวอร์ชันที่ต้องการ จากนั้นคลิก **Download**

## ขั้นตอนการติดตั้ง

<Steps>

### Welcome to Proxmox Virtual Environment

เลือก **Install Proxmox VE** แล้วกดปุ่ม **Enter**

![proxmox-setup-1](/img/proxmox-setup-1.png)

### END USER LICENSE AGREEMENT (EULA)

คลิก **I agree** เพื่อยอมรับข้อตกลงและไปยังขั้นตอนถัดไป

![proxmox-setup-2](/img/proxmox-setup-2.png)

### Proxmox Virtual Environment (PVE)

เลือกดิสก์ที่ต้องการติดตั้ง แล้วคลิก **Next**

![proxmox-setup-3](/img/proxmox-setup-3.png)

### Location and Time Zone selection

- Country: **กรอกประเทศที่คุณอยู่**
- Time Zone: **เลือก Timezone**
- คลิก **Next**

![proxmox-setup-4](/img/proxmox-setup-4.png)

### Administration Password and Email Address

- Password: **กรอกรหัสผ่านที่คุณต้องการใช้สำหรับบัญชีผู้ดูแลระบบ**
- Confirm Password: **กรอกรหัสผ่านซ้ำอีกครั้งเพื่อยืนยัน**
- Email: **กรอกอีเมลสำหรับแจ้งเตือนหรือการกู้คืน**

> [!note]
>
> หากต้องการข้ามขั้นตอนนี้ให้ระบุเป็น `mail@example.local`

![proxmox-setup-5](/img/proxmox-setup-5.png)

### Management Network Configuration

- Hostname: **กรอกชื่อเครื่อง**
- IP Address: **กรอก IP Address**
- Netmask: **กรอก Subnet Mask**
- Gateway: **กรอก Gateway**
- DNS Server: **กรอก DNS Server**

![proxmox-setup-6](/img/proxmox-setup-6.png)

### Summary

คลิก **Install** เพื่อเริ่มการติดตั้ง Proxmox VE

![proxmox-setup-7](/img/proxmox-setup-7.png)

### เข้าใช้งานผ่านเบราว์เซอร์ที่ URL

- เปิดเว็บบราวเซอร์ของคุณ แล้วพิมพ์ **IP Address** ที่แสดงบนหน้าคอนโซล ตามด้วยพอร์ต **8006**

  ```url copy
  https://<ip_address>:8006
  ```

- ใช้ **Username** เป็น `root` และ **Password** ที่คุณตั้งไว้ในขั้นตอนก่อนหน้า

  ![proxmox-setup-8](/img/proxmox-setup-8.png)

</Steps>
