# Zabbix Getting Started

## ขั้นตอนการติดตั้ง

<Steps>

### อัปเดต

```bash
sudo apt update
sudo apt upgrade -y
```

### ติดตั้ง Dependencies

```bash
sudo apt install -y wget curl gnupg
```

### ติดตั้ง Zabbix Repository

```bash
wget https://repo.zabbix.com/zabbix/7.0/ubuntu/pool/main/z/zabbix-release/zabbix-release_latest_7.0+ubuntu24.04_all.deb
sudo dpkg -i zabbix-release_latest_7.0+ubuntu24.04_all.deb
sudo apt update
```

### ติดตั้ง Zabbix Server, Frontend และ Agent

```bash
sudo apt install -y zabbix-server-pgsql zabbix-frontend-php php8.3-pgsql zabbix-nginx-conf zabbix-sql-scripts zabbix-agent2
```

### ติดตั้ง PostgreSQL

```bash
sudo apt install -y postgresql
```

### สร้างฐานข้อมูลและผู้ใช้สำหรับ Zabbix

```bash
sudo -u postgres createuser --pwprompt zabbix
sudo -u postgres createdb -O zabbix zabbix
```

### นำเข้าข้อมูลเริ่มต้น

```bash
zcat /usr/share/zabbix-sql-scripts/postgresql/server.sql.gz | sudo -u zabbix psql zabbix
```

### แก้ไขไฟล์คอนฟิก Zabbix Server

```bash
sudo nano /etc/zabbix/zabbix_server.conf
```

แก้ไขบรรทัด `DBPassword=` ให้เป็นรหัสผ่านที่คุณตั้งไว้สำหรับผู้ใช้ `zabbix`

```bash filename="/etc/zabbix/zabbix_server.conf"
### Option: DBPassword
#       Database password.
#       Comment this line if no password is used.
#
# Mandatory: no
# Default:
# DBPassword=
 
DBPassword=<your_db_password>
```

### ลบไฟล์คอนฟิก Nginx Default

```bash
sudo rm /etc/nginx/sites-enabled/default
```

### รีสตาร์ทและเปิดใช้งาน Services

```bash
systemctl restart zabbix-server zabbix-agent2 postgresql nginx php8.3-fpm
systemctl enable zabbix-server zabbix-agent2 postgresql nginx php8.3-fpm
```

</Steps>

## การเข้าถึง Zabbix Frontend

หลังจากติดตั้งเสร็จสิ้น คุณสามารถเข้าถึง Zabbix Frontend ได้ที่ URL:

```
http://<ip_address>
```

กรุณาเปลี่ยน `<ip_address>` เป็นที่อยู่ IP ของเซิร์ฟเวอร์ที่คุณติดตั้ง Zabbix
